# Hacktoberfest Open Hack Party 2022

## ¿Cómo contribuir?
1. Haz fork de este repositorio
2. Dentro de `content/participants` añade un archivo con tu nombre de usuario en GitHub y extensión `.md`. Ejemplo, si tu nombre de usuario es **monatheoctocat**, crearás un archivo llamado `monatheoctocat.md`.
3. Copia y pega el contenido del archivo `github.md` dentro de tu archivo.
4. Llena el documento con tus propios datos y cuéntanos cuál es **la importancia del open source** para ti.
5. Envía tus cambios a tu propio repositorio usando Git, el UI de GitHub o Codespaces.
6. Regresa al repositorio original y crea un **pull request** con tus cambios.
7. En caso de que tu PR sea inválido, incorpora tus nuevos cambios a tu código y regresa al paso 5.
